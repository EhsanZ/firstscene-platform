# منصة المشهد الأول - نجعل من الكتابة مهنةً أقل وحدة

منصة إلكترونية شاملة تجمع الكتّاب، القرّاء والنقّاد، دور النّشر، والمحرّرين المستقلين في مكان واحد.

## المميزات الرئيسية

### للكتّاب
- تقديم النصوص والمسودات والمخططات
- الحصول على مراجعة مستمرة من الوكالة الأدبية
- إدارة مخطط الكتاب ومسوداته
- إتاحة الأعمال للقراء المراجعين
- الحصول على انطباعات مبدئية
- التواصل مع دور النشر والمحرّرين

### للوكالات الأدبية
- إدارة رحلة الكاتب من الفكرة إلى النشر
- تقييم الأعمال وتقديم ملاحظات شاملة
- تحديد مراحل العمل وحالة التواصل
- إنشاء عقود إلكترونية
- إتاحة الأعمال لدور النشر

### لدور النشر
- الوصول إلى أعمال جاهزة للنشر
- الحصول على انطباعات مبدئية وبيانات تسويقية
- إبداء الاهتمام بالأعمال والتفاوض
- إدارة العقود

### للقرّاء والنقّاد
- قراءة أعمال جديدة مجاناً
- كتابة مراجعات جادة
- المساهمة في تطوير الأدب العربي

### للمحرّرين المستقلين
- اختيار الأعمال للتحرير
- عرض الأسعار والخدمات
- إدارة مهام التحرير

## هيكل المشروع

```
arab-netgally/
├── index.html              # الصفحة الرئيسية
├── login.html              # صفحة تسجيل الدخول
├── register.html           # صفحة إنشاء حساب
├── writer-dashboard.html   # لوحة تحكم الكاتب
├── agency-dashboard.html   # لوحة تحكم الوكالة الأدبية
├── publisher-dashboard.html # لوحة تحكم دار النشر
├── reader-dashboard.html   # لوحة تحكم القارئ/الناقد
├── editor-dashboard.html   # لوحة تحكم المحرر المستقل
├── css/
│   └── main.css           # ملف التنسيقات الرئيسي
├── js/
│   └── main.js            # ملف JavaScript الرئيسي
└── README.md              # هذا الملف
```

## المراحل في رحلة الكاتب

1. **قبل الكتابة** - بناء الفكرة والتخطيط
2. **المسودة الأولى**
3. **إعادة الكتابة والمراجعة والتحرير**
4. **التحرير النهائي وإعداد العمل للنشر**
5. **التفاوض مع دور النشر**
6. **ما بعد النشر**

## حالات التواصل

- جديد
- تم التواصل المبدئي
- جاري تقييم النص/الكاتب
- تم تقييم النص وقبوله بشروط
- تم تقييم النص وقبوله بدون شروط
- تم تقييم النص ورفضه
- انتظار رد من الكاتب
- تم توقيع العقد
- جاهز للنشر
- تم النشر

## التقنيات المستخدمة

- HTML5
- CSS3 (مع دعم RTL للغة العربية)
- JavaScript (Vanilla)
- خط Readex Pro من Google Fonts

## كيفية الاستخدام

1. افتح `index.html` في المتصفح
2. أنشئ حساباً جديداً أو سجّل الدخول
3. اختر نوع الحساب المناسب
4. ابدأ استخدام المنصة حسب دورك
